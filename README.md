# Análise de Vendas com Pandas

Projeto de análise de dados desenvolvido para demonstrar habilidades com a biblioteca Pandas, aplicadas a um conjunto de dados de vendas fictício. O script gera um relatório com receita total, produto mais vendido e produto com maior receita, além de produzir um gráfico de barras para visualização dos resultados.

## Estrutura do repositório

- `vendas.csv`: base de dados utilizada na análise.
- `analise.py`: script principal que processa os dados e exibe o relatório.
- `dashboard.py`: script complementar para geração do gráfico de receita por produto.

## Tecnologias

- Python 3
- Pandas
- Matplotlib

## Como executar

1. Instale as dependências: pip install pandas matplotlib
2. Execute o script de análise: python analise.py
3. Para gerar o gráfico: python dashboard.py

## Resultados esperados

O relatório exibido no terminal inclui:
- Receita total
- Produto com maior volume de unidades vendidas
- Produto com maior receita gerada
- Tabela resumo agrupada por produto

O gráfico é salvo como `dashboard_vendas.png` e também é exibido na tela (se houver suporte).
